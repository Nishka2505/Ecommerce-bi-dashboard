# E-Commerce Business Intelligence Dashboard

Interactive Power BI dashboard analyzing 100k+ e-commerce orders.

## Key Findings
- 7% of orders arrive late
- Late delivery reduces review scores from 4.6 → 1.6 stars
- $758K revenue at risk identified
- 12% of sellers cause 80% of delays (Pareto principle)

## Files
- `ecommerce_dashboard.pbix` - Power BI dashboard
- `notebooks/` - Python data pipeline
- `data/processed/` - Clean CSV files
- `screenshots/` - Dashboard screenshots

## Quick Start
1. Download `ecommerce_dashboard.pbix`
2. Open in Power BI Desktop
3. Use slicers to filter by date, state, or category
